### 01-Forking-And-Cloning

## Forking & Cloning

# What is Forking?
Forking = Making your own copy of someone else’s GitHub repository.
- Used when you don’t have access to edit the original repo.

# Why Fork?
- To contribute to open-source
- To experiment safely
- To customize someone’s project

# Steps to Fork
-  Go to any GitHub repo
-  Click Fork (top-right)
-  Your own copy is created in your account 

# Clone the Forked Repo
Download it to your laptop.

git clone <your-forked-repo-url>

Example:
git clone https://github.com/anantha/my-forked-project.git

# Work Flow
Original Repo → Fork → Clone → Modify → Push → Pull Request
