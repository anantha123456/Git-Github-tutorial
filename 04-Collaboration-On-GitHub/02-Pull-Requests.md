### 02-Pull-Requests

## Pull Requests (PR)

# What is a Pull Request?
A request to merge your changes into another branch or repo (usually main).

- Used when collaborating with others.

# When to Create a PR?
After pushing changes to GitHub

When review/approval is required

# Steps to Create PR

1. Push your branch
git push -u origin feature-login
2. Go to GitHub → Repo
3. Click Compare & Pull Request
4. Add Title + Description
5. Click Create Pull Request

# After Review
- Approve 
- Merge 
- Close  (optional)
