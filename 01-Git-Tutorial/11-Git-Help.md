### 11 - Git Help 
Git Help is used when you don’t remember a command or want to know how a command works.

# 1. Get General Help
git help
OR
git --help

- Shows all Git topics and commands.

# 2. Help for a Specific Command
git help <command>

Example:
git help commit

- Shows how commit works, options, examples.

# 3. Short Help (Quick Info)
git <command> --help

Example:
git status --help

# 4. Quick Manual in Terminal
git <command> -h

Example:
git log -h

- Gives a short summary.

# 5. List All Git Commands
git help -a

- Shows all available Git commands.


# Simple Summary

| What you want      | Command              |
| ------------------ | -------------------- |
| General help       | `git help`           |
| Help for a command | `git help <command>` |
| Short help         | `git <command> -h`   |
| All commands       | `git help -a`        |

# One Line Memory Trick
- help = information
- help <command> = learn command
